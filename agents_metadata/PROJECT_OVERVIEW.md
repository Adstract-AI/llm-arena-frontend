# LLM Arena – Project Overview

## Description

LLM Arena is a simple web-based evaluation platform designed to compare the performance of different large language models (LLMs) through human feedback.

The system allows users to submit a prompt and receive two anonymous responses generated by two different models. The user then selects which response is better. This enables blind, pairwise comparison of models based on real user preferences.

The main goal of this project is to evaluate **Macedonian fine-tuned LLMs**.

---

## Core Concept

- A user submits a prompt.
- The backend sends the same prompt to two different LLMs.
- The frontend displays both responses as **Answer A** and **Answer B** (without revealing model identities).
- The user selects:
  - A is better
  - B is better
  - Tie / both are equal
- The result is stored for later analysis.

This setup ensures **blind evaluation** and reduces bias.

---

## Tech Stack

- **Frontend**: React (simple UI for prompt input, response display, and voting)
- **Backend**: Django (API, business logic, model orchestration, data storage)
- **LLM Providers**:
  - OpenAI
  - Anthropic
  - Macedonian fine-tuned models (available via OpenAI-compatible API)

---

## System Architecture

### Frontend
- Prompt input field
- Submit button
- Two response panels (A and B)
- Voting buttons
- Loading and error states

### Backend
- Receives prompt and initializes a "battle"
- Selects two models (randomly or predefined)
- Queries both models with the same prompt
- Stores responses and metadata
- Returns anonymized responses to frontend
- Accepts and stores user votes

---

## Key Requirements

- **Blind comparison**: Users must not know which model generated which response. It should be shown only after tha vote.
- **Same prompt for both models**: Ensures fairness
- **Randomized response order**: Prevents positional bias
- **Persistent storage**: All prompts, responses, and votes must be saved

---

## Goal

The system provides a simple but effective way to:
- Collect human preference data
- Compare local (Macedonian) vs global LLMs
- Analyze performance trends across models

This project focuses on evaluation, not conversation or chat history.
